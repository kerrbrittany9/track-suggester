Check out Epicodus. It's a pretty sweet school to learn to code and it offers multiple courses! Find out which one is right for you by taking the survey below.

CSS and Design Track

Looks like you've got the faculties for front-end development! Maybe you were considered the Picasso of the neighborhood or you just really have an eye for color. In any case, the CSS course will teach you to use those skills to make beautiful web pages. The course focuses on HTML to build the structure of the page and CSS, which allows you create visually pleasing webpages!

C# Track

Backend and business ninja here you go! Studying C# will give you the tools to work for the likes of Microsoft and other large companies as well as government agencies. This course includes object-oriented design basics, behavior-driven development and database basics so that you will feel confident in solving the more complex programming problems.

Ruby Track

You are the perfect fit for Ruby. As we at Epicodus put it, Ruby focuses on simplicity and productivity. It is most commonly used in startups or fresh new companies: an environment that champions building exciting web applications.
